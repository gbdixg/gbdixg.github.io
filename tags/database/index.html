Config Manager 'Potential SQL performance issue caused by change tracking retention period'

2019-02-23 3 min read memcm SCCM Geoff
If you come across this warning during a Config Manager Current Branch site upgrade, read on for the solution. Warning - Potential SQL performance issue caused by change tracking retention period. This warning was apparently added from CM current branch 1810 or later. Running the pre-requisite check CM Current Branch updates are initiated in the Administration > Updates and Servicing node of the CM console. The pre-requisiste check option will highlight any errors and warnings before the upgrade starts. Continue reading
